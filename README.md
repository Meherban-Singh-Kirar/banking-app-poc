## Banking Application POC

This project is a proof of concept (POC) banking application designed to demonstrate
core banking functionalities and modern application development practices.

### Features
- User registration and authentication (JWT)
- Account creation and management
- Fund transfer between accounts
- Transaction history
- Role-based access control

### Tech Stack
- Frontend: React
- Backend: Spring Boot
- Security: JWT Authentication
- Database: MySQL
- Build Tool: Gradle

### Purpose
- Hands-on learning
- Full-stack development practice
- Interview and portfolio showcase
